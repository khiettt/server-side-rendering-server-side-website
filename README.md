## Beschrijving

Als je opzoek bent naar een cadeau voor iemand, is milledoni de website voor je zoektocht. Ze hebben cadeautjes voor alle leeftijden en gelegenheden en je kan samen met de chatbox de juiste cadeau vinden. Milledoni wilde een vernieuwede categorie pagina met de nieuwe huisstijl/design die wij van hun hebben gekregen.

## Gebruik
Ik wilde voor milledoni's categorie pagina meer kleur geven en de design een beetje veranderen. 

[Milledoni's](https://milledoni.com/gb/en) verniewde website filtert via de chatbox. Aangezien een chatbox maken wat lastiger is voor ons hebben wij de taak gekregen om een filter knop te maken. 

**Ik weet dat Milledoni voor een gele kleur hadden gekozen omdat zij een warme/verwelkomende sfeer wilde creëren voor de categorie pagina ben ik gaan expirimenteren met groen omdat zij verschillende logo's hadden in verschillende kleuren en groen leek mij een leuk idee.**
